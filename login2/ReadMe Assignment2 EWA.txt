How to Run The Application-

1) Install MySql Db, MongoDb, Java, Text Editor.
2) Go to C:\apache-tomcat-7.0.34\webapps\login2\WEB-INF\classes
3) Compile the Source Code Using Java
4) Go to Google Chrome and type- http://localhost/login2/
5) Login Menu Will Appear
6) For Admin I have made Database table to store his username and password. USERNAME-admin PASSWORD-admin.
7) For Client, he/she can sign up, I have made few entries in database table called registeration.
8) For SalesPerson I have made Database table to store his username and password. USERNAME-sales PASSWORD-sales

Note- code for executing MYSql queries are in file MySQLDataStore.java
